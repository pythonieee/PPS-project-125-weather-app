### This project is a simple and interactive Weather App developed using Streamlit and the WeatherAPI.
### The app allows users to enter a city name and get real-time weather information such as temperature, humidity, wind speed,
### and weather conditions. Streamlit was used to create a clean and user-friendly interface, while WeatherAPI provided accurate and up-to-date weather data.
### The app is lightweight, easy to use, and can be further improved with features like forecasts, weather alerts, and multi-language support.
